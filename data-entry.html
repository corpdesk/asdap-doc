Statistics Wizard - User Manual

The Statistics Wizard is a step-by-step tool to create, manage, and submit cooperative data efficiently. Follow the steps below to complete your tasks.

Step 1: Select or Create a Reference

Each data batch must be linked to a reference. You can either select an existing reference or create a new one.

1.1 Select an Existing Reference

Use the dropdown to select an existing reference. Upon selection, you can view its details:

Field Description
ID The unique identifier for the reference.
Name The name of the reference.
1.2 Create a New Reference

If no existing reference meets your needs, create a new one by providing the following details:

  1. Name: Enter a name for the reference.
  2. Description: Provide a description for the reference.
  3. Click Save Reference to proceed or Cancel to discard changes.
Step 2: Datasource Information

Specify the source type, geographic scope, and time period for your data.

2.1 Select Data Source Type

Use the dropdown to choose the source type (e.g., `Continental SACCO`, `National Apex`).

2.2 Configure Geographic Information

Based on the selected source type, provide additional geographic details:

Parameter Description
Continent Specify the continent for `Continental SACCO` types.
Country Select a country for `National Apex` or `Microfinance SACCO` types.
SubRegion Optionally specify a subregion if required.
2.3 Configure Temporal Information

Specify the time period using one of the following options:

  • Year: Enter a specific year.
  • Date Selector: Use a calendar input to select a specific date.
Step 3: Statistical Data

Provide detailed statistical data for the cooperative, including:

  • Number of SACCOs: Total SACCOs in the dataset.
  • Number of Members: Total members in the SACCOs.
  • Saves & Shares: Savings and shares in USD.
  • Loans: Loans issued in USD.
  • Assets: Total assets in USD.
  • Penetration: Membership penetration percentage.
  • Reserves: Reserves in USD.
  • WOCCU Membership: Indicate if the SACCO is a WOCCU member.

Use the wizard's built-in validation tools to ensure data accuracy. Submit the final data for processing at the end of the wizard.
