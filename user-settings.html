User Settings

User settings is divided into three key sections to help you manage your account effectively:

  • Personal Information
  • Notifications
  • Account Security
Personal Information

The personal information section allows you to manage the following:

Control Usage
Change Password Update your current password or initiate a "Forgot Password" process.
Multi-Factor Authentication Enhance account security by configuring multi-factor authentication (MFA) using email, phone, or authenticator apps.
Language and Region Set your preferred language and region for a customized experience.
Profile Picture Upload or update your profile picture.
Notifications

Receive and track notifications from various system modules and features, including:

Control Usage
System Alerts Receive warnings and critical notifications from the system.
Admin Messages View important messages sent by system administrators.
Chat Messages Get notifications for new messages received via the chat system.

The notifications list highlights:

  • Unread notifications with a flag to help you identify them.
  • Notifications requiring action, along with their current status.
Account Security

Monitor and manage your account security with the following tools:

Control Usage
Activity Logs View detailed logs of recent account activities, including login attempts and device access.
Active Devices Monitor devices currently logged into your account.
Session Management Log out or block specific devices directly from this interface to secure your account.

Keeping your settings up to date helps ensure your account remains secure, personalized, and efficient.
